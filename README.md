# Green Coding Sustainability Research

Independent research project investigating:

**"To what extent can green coding contribute to global sustainability goals in the digital age?"**

## Repository Structure

- `paper/` → Final research paper (PDF)
- `code/` → Python programs used in the analysis
- `graphs/` → Charts and figures generated for the study

## Research Areas

- Green Software Engineering
- Sustainable Computing
- Energy-Efficient Algorithms
- Carbon Footprint Reduction
- Global Sustainability Goals
